1. Detalhes da Interface

Este projeto é um conjunto de telas para painel administrativo (Dashboard) com design consistente e otimizado para o usuário.

Inclui telas de formulários (cadastro), grids de dados e vários componentes. O projeto possui variações de layout, posicionamento e cores , servindo como inspiração para ser utilizado de acordo com cada tela e conteúdo.

Responsivo

O layout se adapta perfeitamente a todos os tamanhos de tela (desktop, tablet e mobile).

Design Moderno

Paleta de cores atualizada com foco em contraste, hierarquia visual e acessibilidade, facilitando a identificação rápida das informações.

Bootstrap & Customização

Desenvolvido em Bootstrap 5 . Sidebar e Header são customizáveis, preparados para refletir a identidade visual da sua empresa, podendo ser alterados conforme a cor da marca (temas, cores e logo).

2. Estrutura Modular

O código está dividido em componentes para facilitar a integração e evitar a repetição de código em um ambiente de desenvolvimento MVC (Model-View-Controller) ou com uso de *Includes*:

  • Header (Topo)

    Barra superior fixa contendo o menu mobile, busca e notificações.

  • Sidebar (Menu)

    Menu lateral responsivo, que colapsa no desktop e se torna um offcanvas no mobile.

  • Main Content (Conteúdo Principal)

    Área principal onde o conteúdo de cada tela deve ser inserido (o corpo da View).

  • Footer (Rodapé)

    Bloco padrão, contendo informações de direitos autorais e links secundários/de apoio.

Dica de Integração

As telas dashboard.html e as demais compartilham o mesmo layout principal (Header, Sidebar e Footer), bastando extrair os componentes para uso no sistema com a utilização de Includes e partials.
